Welcome to Sample JSF Web Application

This is a sample application used by the DevSecOps-Hosting and Provisioning team for testing and debugging purposes.

This app is built using following tech stack:

  • JSF 2.3 (Designing web pages)
  • MongoDB 4.2 (Backend persistent storage)
  • Java 8 (Backend services, connecting UI with DB)
  • Apache Tomcat 9.0 (Web server)
  • Primefaces 3.4 (provides additional CSS for JSF web pages)
